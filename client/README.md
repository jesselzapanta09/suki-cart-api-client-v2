# MSIT 114 - Activity 17 , 19, 21, 23 Frontend/Client

This project is the **client-side** application for the RailManager system.

## Installation & Setup

### Step 1: Clone the Repository
```bash
git clone https://github.com/jesselzapanta09/railmanager.git
cd railmanager
```

### Step 2: Install Dependencies
```bash
npm install
```

### Step 3: Run the Development Server
```bash
npm run dev
```

The App should now be running at `http://localhost:3000/railmanager/`.
